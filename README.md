# Pet Risk Prediction

## Overview
This notebook demonstrates how to predict pet health risk values using various machine learning and deep learning techniques. It covers data preprocessing, feature engineering, model training, evaluation, and comparison of multiple regression models on pet records data.

## Features
- Data preprocessing and missing value handling for pet records
- Feature engineering including age conversion, breed frequency, and note text analysis
- Training and evaluation of multiple traditional regression models (Linear Regression, ElasticNet, Random Forest, XGBoost, CatBoost, etc.)
- Implementation of a deep learning regression model using TensorFlow Keras
- Model performance comparison with key regression metrics

## Tech Stack
- Python
- Jupyter Notebook / Google Colab
- pandas, numpy, scikit-learn
- xgboost, lightgbm, catboost
- tensorflow / keras

## How to Use
1. Clone or download the repository.
2. Place your pet records CSV dataset in the specified path or update the path in the notebook.
3. Run the notebook cells sequentially in Jupyter or Google Colab.
4. Analyze the output model performance metrics and choose the best model for your needs.
